Creation

 Boxes can be created by pressing the Ctrl key while clicking and releasing the mouse
at the wished dimensions of the box.
: Sound Box : Ctrl+Shift+S : Switch current interaction to Sound Box creation
: Control Box : Ctrl+Shift+C : Switch current interaction to Control Box creation
: Parent Box : Ctrl+Shift+B : Switch current interaction to Parent Box creation

Edition

 : Copy : Ctrl+C : Copy a selection of boxes
: Cut : Ctrl+X : Cut a selection of boxes
: Paste : Ctrl+V : Paste a selection of boxes

View

 : Zoom In : Ctrl+'+' : Process a Zoom In
: Zoom Out : Ctrl+'-' : Process a Zoom Out
: Edit Attributes : Ctrl+E : Open/close attributes editor
For further information about editor, see "Editor" tab.

Sound
: Specific Play : No Shortcut : Sounds will be played in boxes selected modes
: Direct Play : No Shortcut : Sounds linked to sound boxes will be played
: Synth Play : No Shortcut : Sounds synthetised from sound boxes attributes will be played
Acceleration : No Shortcut : Select acceleration for play
: Play : Space : Play the scene
: Stop : Space : Stop playing

Network



: Network : No Shortcut : Tune network attributes
